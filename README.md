# Luz & Lente – Site de Fotografia

Projeto front-end de um site institucional para serviço de fotografia, desenvolvido com foco em layout personalizado, animações e experiência visual.

## Demo

Projeto publicado utilizando GitHub Pages:

https://thiagorod-48.github.io/Fotografia-site-/

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar a construção de layouts responsivos utilizando HTML e CSS (com Sass), sem o uso de frameworks CSS, além da aplicação de animações com JavaScript utilizando a biblioteca GSAP.

A proposta foi criar uma interface simples, visualmente agradável e funcional, explorando fundamentos de front-end.

## Funcionalidades

- Página institucional para serviço de fotografia
- Menu responsivo com JavaScript
- Animações de entrada utilizando GSAP
- Botão de contato via WhatsApp
- Layout responsivo sem uso de frameworks CSS

## Tecnologias utilizadas

- HTML5
- CSS3
- Sass
- JavaScript
- GSAP (GreenSock Animation Platform)

## Como executar o projeto

Basta abrir o arquivo `index.html` em um navegador web.

Não é necessária instalação de dependências ou servidor local.

## Observações

Projeto desenvolvido exclusivamente em front-end, com foco em layout, animações e fundamentos de desenvolvimento web.
